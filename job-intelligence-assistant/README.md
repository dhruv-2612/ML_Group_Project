# Job Market Intelligence Assistant

An AI-powered career guidance chatbot that integrates ML clustering, RAG (Retrieval Augmented Generation), and LLMs to provide personalized job market insights for MBA students.

## Features
- Web scraping job postings (Naukri.com/Instahyre)
- ML Analysis: K-Means clustering & Salary prediction
- RAG System: LangChain + Google Gemini + ChromaDB
- Interactive Streamlit Dashboard
